Recommended Program: Information Technology - Programming, AAS

Why This Program for AI?

  • Core Programming Skills: Hands-on experience in multiple programming languages
  • Analytical Thinking: Problem-solving skills essential for AI model design
  • Mathematics Foundation: Statistical and computational mathematics for AI
  • Data Management: Database skills for AI training datasets

Program Details

Total Credit Hours: 60
Program Type: Associate of Applied Science
Typical Duration: 2 Years
Available Locations: Multiple Campuses

Foundational Tier (First Year)

Essential computing, mathematics, and communication skills

COSC 1301 3 Credits

Introduction to Computing

Overview of hardware, OS, application software. Builds digital literacy for programming and data work.

Prerequisites: None
COSC 1436 4 Credits

Programming Fundamentals I

Structured programming: data types, control structures, functions, arrays, debugging. Teaches core coding logic.

Prerequisites: College-level math
MATH 1314 3 Credits

College Algebra

Algebraic functions, equations, and problem-solving. Foundation for advanced mathematics in AI.

Prerequisites: TSI Math requirements
MATH 1342 3 Credits

Elementary Statistical Methods

Probability, distributions, descriptive statistics, hypothesis testing. Foundation for evaluating ML algorithms.

Prerequisites: College-level math
ENGL 1301 3 Credits

Composition I

Written communication skills essential for technical documentation and AI project reporting.

Prerequisites: TSI Writing requirements
BCIS 1305 3 Credits

Business Computer Applications

Spreadsheet and database applications for data analysis and business intelligence.

Prerequisites: Basic computer literacy

Core Tier (Second Year)

Advanced programming, data management, and system design

COSC 1437 4 Credits

Programming Fundamentals II

Object-oriented programming, classes, algorithm analysis. Prepares students for modular AI code development.

Prerequisites: COSC 1436
COSC 2436 4 Credits

Programming Fundamentals III

Data structures: stacks, queues, trees, graphs. Fundamental for implementing efficient AI algorithms.

Prerequisites: COSC 1437
ITSE 2417 4 Credits

Java Programming

Object-oriented programming in Java. Important for AI frameworks and enterprise applications.

Prerequisites: Programming experience
ITSE 2409 4 Credits

Database Programming

SQL and database design. Essential for managing and querying AI training datasets.

Prerequisites: Programming fundamentals
MATH 2413 4 Credits

Calculus I

Limits, derivatives, integrals. Underlies optimization algorithms used in training AI models.

Prerequisites: MATH 2412 or equivalent
ITNW 1425 4 Credits

Fundamentals of Networking Technologies

Network principles for distributed AI systems and cloud computing applications.

Prerequisites: Basic computing knowledge

Advanced Tier (Specialization)

Specialized programming and system analysis for AI applications

ITSE 1450 4 Credits

System Analysis and Design

Systems thinking and design methodology essential for AI solution architecture.

Prerequisites: Programming experience
ITSE 1479 4 Credits

Introduction to Scripting Languages

Scripting for automation and data processing, often including Python - key for AI development.

Prerequisites: Basic programming
ITSE 1430 4 Credits

Introduction to C# Programming

Modern programming language for enterprise AI applications and Microsoft AI frameworks.

Prerequisites: Programming fundamentals
ITSE 1411 4 Credits

Beginning Web Programming

Web development skills for building AI-powered web applications and interfaces.

Prerequisites: Basic programming
ITSE 1333 3 Credits

Mobile Applications Development

Mobile app development incorporating AI features and machine learning capabilities.

Prerequisites: Programming experience

Fundamentals of Information Security

Security principles crucial for protecting AI systems and sensitive training data.
